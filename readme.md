# API de Star Wars

<p>Desde esta API REST se pueden solicitar datos de personajes, planetas, armas y naves de Star Wars</p>

### Dependencias

<ul>
    <li>NodeJS</li>
    <li>Express</li>
    <li>Cors</li>
    <li>Mongoose</li>
    <li>DotEnv</li>
    <li>Morgan</li>
</ul>

</ul>
